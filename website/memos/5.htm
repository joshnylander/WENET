WC Jail Demo - WENET Service

We have implemented a WENET service point based on the limited data available through the Jail Roster application. Basing our demo service point on an existing public application means that we can make the service publicly available for testing. This demo may not be a complete implementation, see noted below. The Jail roster is a list of the persons currently in the Whatcom County Jail. The service provides the person's full name, booking number and date booked.

20040317 - Implemented query and document retrieval. To test, try the following URLs:

  • To find all bookings of persons currently in Jail: http://www.co.whatcom.wa.us/apps/wenet/wcjaildemo/query.jsp?query=
  • To find a specific booking number based on BookingFileID: http://www.co.whatcom.wa.us/apps/wenet/wcjaildemo/query.jsp?query=/wenet:document/jxdd3:Booking/jxdd3:BookingFileID[jxdd2:ID='154916']
  • To find a specific booking number based on ActivityID: http://www.co.whatcom.wa.us/apps/wenet/wcjaildemo/query.jsp?query=/wenet:document/jxdd3:Booking/jxdd3:ActivityID[jxdd2:ID='154916']
  • To find a booking by person name: http://www.co.whatcom.wa.us/apps/wenet/wcjaildemo/query.jsp?query=/wenet:document/jxdd3:Booking/jxdd3:BookingSubject/jxdd3:PersonName[jxdd3:PersonFullName='ANASKAN']
  • To find a bookings by date: http://www.co.whatcom.wa.us/apps/wenet/wcjaildemo/query.jsp?query=/wenet:document/jxdd3:Booking[jxdd3:ActivityDate='2004-03-08']
