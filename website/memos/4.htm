WENET Schema 1

The WENET project has defined a schema, called WENET Schema to complement the WENET Protocol.

Schema: http://www.whatcomcounty.us/apps/wenet/schema/wenet/wenet_schema_1.xsd

As of this date the schema is in draft form. It should be expected to change in two ways. The first will be to improve documentation, code re-use, schema complience and to better limit options to reduce errors. These changes should not effect the implementation of services based on the schema. The second way the schema should be expected to change is to increase the number of supported documentTypes. The wenet:document currently defines 3 types of documents and should serve as an example of how documentTypes are used. It is anticipated that this number will climb to imclude a more complete representation of the data available in our RMS systems.
