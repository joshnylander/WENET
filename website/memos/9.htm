WENET Schema 1.2

The WENET project has defined a schema, called WENET Schema v1.2 to complement the WENET Protocol.

Schema: http://www.whatcomcounty.us/apps/wenet/schema/wenet/wenet_schema_1-2.xsd

Schema Guidelines:

In addition to the schema, implemetations MUST adhere to the following guidelines in deploying the schema.

  • use of ID/RefID for linking elements

Changes:

-11/23/2004: Added ID/RefID language. Marked document "FINAL". Updated date.

-11/1/2004: Released new schema version 1.2 with major and incompatible changes from version 1.1. Most notably, removed the requirement that instances be based on the GJXDM and removed references to the GJXDM.

- 5/18/2004: Released new schema version 1.1 with major and incompatible changes from version 1.

Major Changes:

  1. Made changes to support changes in the WENET Protocol.
  2. Made changes to make it more compatible with the GJXDM v3.0.
