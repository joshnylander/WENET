WENET

The WENET project has one primary goal

Design and deploy a computerized process that allows the sharing and/or exchanging of information between all law and justice practitioners, within the county’s justice enterprise, that tracks offenders from the moment they enter the system until re-integrated back into the community.

Specifically we will deploy services that allow for inquiry into each agencies records management systems via the developed WENET Protocol using the Global Justice XML Data Model and connecting all local agencies.

New: Information for developers can now be found in the Developers Corner and on the site for the November 04 and April 05 Developers Workshop.
